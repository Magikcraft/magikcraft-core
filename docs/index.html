@magikcraft/core

Core Magikcraft utilities for SMA plugins

Installation

npm install @magikcraft/core

Testing

Use the SMA image in test mode to run the unit tests.

Make sure you have the latest version of smac installed:

npm i -g smac

Then run the tests:

npm test
